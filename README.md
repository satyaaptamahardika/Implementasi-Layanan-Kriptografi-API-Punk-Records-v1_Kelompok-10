# Implementasi-Layanan-Kriptografi-API-Punk-Records-v1_Kelompok-10
Security Service API untuk Punk Records-v1 sebagai proyek UAS Keamanan dan Integritas Data. Dibangun dengan FastAPI dan library cryptography untuk menyimpan public key, memverifikasi digital signature Ed25519 dan EC SECP256K1, melakukan integrity check PDF, serta relay pesan aman.
